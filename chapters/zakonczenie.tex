\chapter{Podsumowanie}
Pomyślnie zrealizowano wszystkie założone cele projektu. Wytworzono moduły detektora, serwera API oraz serwera webowego. Detektor rozpoznaje zdarzenia na nagraniu testowym, prawidłowo animizuje twarze i zamazuje twarze. Zaistniał problem rozdzielczości kamery tj. moduł detektora wymagał dopasowania hiperparametrów do nagrań z kamery testowej, aby móc dalej prawidłowo wykrywać obiekty na nagraniach.

\section{Dalsze możliwości rozwoju}
System ma charakter prototypu. Wiele rzeczy można w nim pozmieniać i zoptymalizować. Można:
\begin{enumerate}
\item Dodać więcej wykrywanych obiektów i zdarzeń;
\item Zoptymalizować pętlę przetwarzania klatek nagrań;
\item Zoptymalizować YOLO, aby działało w rozsądnym czasie na nagraniach z monitoringu PW, poprzez lepsze dopasowanie hiperparametrów lub destylację wiedzy w sieci YOLO.
\item Dodać mechanizm logowania i autentykacji do interfejsu użytkownika
\item Zabezpieczyć punkty końcowe serwera API przed nieupoważnionym dostępem
\item Rozbudować mechanizm detekcji zdarzeń o grę terenową.
\end{enumerate}
